## 🔧 配置说明

### 环境变量
创建 `.env` 文件：
```bash
VITE_API_BASE_URL=http://localhost:3000
```

### 启动项目
```bash
npm run dev
```

## 📝 使用说明

所有页面现在都使用真实的后端 API：
- 数据从后端获取
- 创建/更新/删除操作调用后端接口
- 自动处理 Token 认证
- 统一的错误处理
